== Introduction

Ce dépot contient le code de notre projet **Prévision Météo en Australie - MLOps**, développé durant notre cursus {DSMLOps} chez {DataScientest}.

Ce projet a pour objectif le déploiement et le monitoring d'une solution pour la prédiction météo selon un cycle MLOps.

Le but est de prévoir s'il va pleuvoir le lendemain.

Basé sur les données initiales d'un projet {KagglePrj}.

Notre travail :

* définir un process MLOps.
